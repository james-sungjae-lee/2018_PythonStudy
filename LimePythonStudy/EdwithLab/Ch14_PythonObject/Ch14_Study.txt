Ch14 : 파이썬 오브젝트

Part1 : Object in Python

메소드 : 클래스 내의 함수
필드 / 속성 : 클래스 내의 변수


Part2 : Object & Class

클래스 : 템플릿
객체 : 인스턴스

x = list()
type(x) : x 가 어떤 클래스인지 알려준다.
dir(x) : x 가 속한 클래스의 모든 메소드 / 필드를 보여준다.

y = 'Hello, world!'
y 는 string 클래스이다.
즉, dir(y) 를 작성하면 string 클래스 내의 다양한 메소드 / 필드를 볼 수 있다


Part3 : Object Life Cycle

라이프 싸이클 : 생명주기
= 객체의 생성 - 삭제의 반복을 의미한다.

생성자 : 객체의 생성에 사용됨
> 객체를 정의할 때 자동으로 사용

소멸자 : 객체의 소멸에 사용됨
> 특정 변수를 덮어쓸 때 등에 자동으로 사용
